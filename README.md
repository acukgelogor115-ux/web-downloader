# 🎵 YouTube Downloader Pro (Static Web)

Aplikasi web sederhana untuk mengunduh video atau audio dari YouTube secara gratis.

## ✨ Fitur Utama
- **Tanpa Backend:** Berjalan 100% di GitHub Pages.
- **Multi-Format:** Video (MP4) & Audio (MP3).
- **Cepat & Ringan:** Responsif untuk mobile.

## 🚀 Cara Penggunaan
1. Paste link YouTube.
2. Pilih format.
3. Klik "Cari Video".
4. Klik "Download Sekarang".

## ⚠️ Catatan
- Gunakan Secure DNS jika API terblokir ISP.
- Hanya untuk edukasi & penggunaan pribadi.

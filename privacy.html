📄 Kebijakan Privasi

Terakhir diperbarui: 29 Juni 2026

Ringkasan Singkat: Kami tidak mengumpulkan, menyimpan, atau menjual data pribadimu. Website ini 100% gratis dan tidak memerlukan registrasi.

1. Pendahuluan

Kebijakan Privasi ini menjelaskan bagaimana Downloader Musik ("kami", "situs") mengelola informasi saat kamu menggunakan layanan kami di https://acukgelogor115-ux.github.io/web-downloader/.

2. Informasi yang Kami Kumpulkan

Kami TIDAK mengumpulkan data pribadi apapun. Secara spesifik:

  • ❌ Tidak ada registrasi atau akun pengguna
  • ❌ Tidak ada penyimpanan riwayat download
  • ❌ Tidak ada cookie pelacakan
  • ❌ Tidak ada analitik pihak ketiga (Google Analytics, dll)
  • ❌ Tidak ada log IP address

3. Data URL YouTube

Saat kamu memasukkan link YouTube untuk diunduh:

  • URL tersebut hanya diproses sementara oleh server API pihak ketiga (Cobalt Community).
  • Kami tidak menyimpan URL tersebut di database manapun.
  • Setelah file dikirim ke browser-mu, data URL langsung dihapus dari memori server.

4. Pihak Ketiga

Situs ini menggunakan layanan pihak ketiga untuk memproses download:

  • Cobalt Community API - Server open-source yang memproses request download. Mereka juga memiliki kebijakan privasi masing-masing dan tidak menyimpan log pengguna.
  • GitHub Pages - Tempat website ini di-hosting. GitHub mungkin mengumpulkan log akses standar (IP, browser) sesuai kebijakan privasi mereka.

5. Keamanan

Semua komunikasi antara browser-mu dan server kami dilakukan melalui koneksi HTTPS terenkripsi, sehingga data yang dikirimkan aman dari intersepsi pihak ketiga.

6. Hak Cipta & Tanggung Jawab

Downloader Musik adalah alat teknis semata. Pengguna bertanggung jawab penuh atas:

  • Memastikan konten yang diunduh tidak melanggar hak cipta.
  • Menggunakan file hasil download sesuai hukum yang berlaku di wilayahnya.
  • Kami tidak bertanggung jawab atas penyalahgunaan layanan ini.

7. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini sewaktu-waktu. Perubahan akan ditampilkan di halaman ini dengan tanggal "Terakhir diperbarui" yang baru.
